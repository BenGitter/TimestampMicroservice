Timestamp Microservice API

This API takes in a natural date (e.g. "August 15, 2016") or an Unix timestamp (e.g. "1471219200") and will return a JSON response with both the natural date and the Unix timestamp.

In case of an invalid date or timestamp, the response will set both the date and timestamp property to null.

Example usage

https://project-name.herokuapp.com/August%2015,%202015
https://project-name.herokuapp.com/1471219200

Example response

{ "unix": 1471219200, "natural": "August 15, 2016" }
